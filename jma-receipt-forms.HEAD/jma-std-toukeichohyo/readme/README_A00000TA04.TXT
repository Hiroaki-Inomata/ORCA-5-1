＜インストール方法及び実行環境について＞
  １．ソースのコピーとコンパイル
      以下のインストールシェルをrootユーザで実行して下さい。
      bash install.sh

  ２．プログラムオプションのインストール
      初期インストールの場合、/prgoption/のディレクトリに移動し、
      次のインストールシェルをrootユーザで実行してください。

      bash  prgoption.sh <医療機関識別番号>
      例). bash  prgoption.sh 1

      インストール後、年齢情報を「91 マスタ登録」「101 システム管理マスタ」
     「1910 プログラムオプション情報」へ設定し登録してください。

  ３．プログラムのメニュー登録
     「91 マスタ登録」、「101 システム管理マスタ」より
     「3002  統計帳票出力情報(月次)」に対して以下のプログラム登録の処理を行う。

     ＜プログラム登録設定情報＞
       帳票番号      ：任意に設定
       帳票名        ：来院・在院状況分析表
       プログラム名  ：A00000TA04
       帳票パラメタ１：開始年月  YM  （必須）  集計対象の開始年月を入力します。
       帳票パラメタ２：入外区分  FREE（必須）  1：入院  2：外来
       帳票パラメタ３：発行区分  FREE（任意）  空白、1：診療科別  2：医師別  3：年齢別
