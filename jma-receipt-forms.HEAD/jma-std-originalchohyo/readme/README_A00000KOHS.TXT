＜プログラムのメニュー登録＞
  「91 マスタ登録」、「101 システム管理マスタ」より
  「3002  統計帳票出力情報(月次)」に対して以下のプログラム登録の処理を行う。

  プログラム登録設定情報
    帳票番号      ：任意に設定
    帳票名        ：償還払公費患者一覧表
    プログラム名  ：A00000KOHS
    帳票パラメタ１：請求年月  、YM

  「42 明細書」でレセプト作成を行った後に起動してください。

＜プログラムオプションの設定＞
  必要に応じて「91 マスタ登録」「101 システム管理マスタ」
  「1910 プログラムオプション情報」へ設定し登録してください。
