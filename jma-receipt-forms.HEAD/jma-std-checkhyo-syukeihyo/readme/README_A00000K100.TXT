＜プログラムのメニュー登録＞
  「91 マスタ登録」、「101 システム管理マスタ」より
  「3002  統計帳票出力情報(月次)」に対して以下のプログラム登録の処理を行う。

  ・プログラム登録設定情報
    帳票番号      ：任意に設定
    帳票名        ：診療行為区分別集計表
    プログラム名  ：A00000K100
    帳票パラメタ１：出力帳票      、FREE、"任意"にチェック
    帳票パラメタ２：集計年月      、YM  、"必須"にチェック
    帳票パラメタ３：入外区分      、FREE、"必須"にチェック
    帳票パラメタ４：保険適用外    、FREE、"任意"にチェック
                    0の時、保険適用外を集計に含める。
                    未入力の時は保険適用外を集計に含めない。
    帳票パラメタ５：包括区分      、FREE、"任意"にチェック
                    0の時、包括分を集計に含める。 ※保険別は包括欄へ集計されます。
                    未入力の時は包括分は集計に含めない。
    帳票パラメタ６：表示区分      、FREE、"任意"にチェック
                    1:点数表示、2:金額表示
                    ※上記以外は点数表示になります。
