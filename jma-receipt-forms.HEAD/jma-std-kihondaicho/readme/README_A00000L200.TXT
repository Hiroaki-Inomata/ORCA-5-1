＜インストール方法及び実行環境について＞
  １．ソースのコピーとコンパイル
      以下のインストールシェルをrootユーザで実行して下さい。
      bash install.sh

  ２．プログラムのメニュー登録
     「91 マスタ登録」、「101 システム管理マスタ」より
     「3002  統計帳票出力情報(月次)」に対して以下のプログラム登録の処理を行う。

     ＜プログラム登録設定情報＞
      帳票番号      ：任意に設定
      帳票名        ：保険別患者台帳
      プログラム名  ：A00000L200
      帳票パラメタ１：対象年月、      YM(必須)
      帳票パラメタ２：入外区分、      FREE(任意) 「1」入院、「2」外来
      帳票パラメタ３：保険者番号、    FREE(任意)  (注４)
      帳票パラメタ４：公費負担者番号、FREE(任意)  (注４)
      帳票パラメタ５：本人・家族、    FREE(任意) 「1」本人、「2」家族
      帳票パラメタ６：診療科、        FREE(任意)  旧総合病院時のみ
      帳票パラメタ７：印字順          FREE(任意) 「0」カナ氏名「1」保険者番号「2」公費負担者番号
                                                  未入力は患者番号順です

      (注１) 実行環境へのモジュールのコピーは必要ありません。
      (注２) /usr/lib/jma-receipt/lddef/orcabt.bdファイルにプログラム情報を追加する必要はありません。
      (注３) プログラムオプションの項目SETTEIが２の時は、負担者番号と、受給者番号をチェック、２以外は、
             負担者番号のみチェックします。(デフォルトは１です。)
      (注４) "*"と"＊"は同じ桁数の全数字、文字(全角のみ)を対象にします。但し、スペースは対象外です。
             未入力の場合は、該当するデータが未入力の患者を対象にします。
             先頭に"@"と入力すると、データが設定されている患者を対象にします。
